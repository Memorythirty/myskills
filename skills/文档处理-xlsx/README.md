# 文档处理-xlsx

## 作用
Excel 电子表格处理，支持：
- 创建新电子表格（含公式和格式）
- 读取或分析数据
- 修改现有电子表格（保留公式）
- 数据分析和可视化
- 重新计算公式

## 安装来源
- 仓库：appautomaton/document-SKILLs
- 链接：https://github.com/appautomaton/document-SKILLs

## 安装方式
```bash
npx skills add appautomaton/document-SKILLs -g -y
```

## 指令
- `/xlsx` - 处理 Excel 电子表格
