# 学习工具-ship-learn-next

## 作用
将学习内容（如 YouTube 转录、文章、教程）转化为可执行的实施计划，使用 Ship-Learn-Next 框架：
- 将建议、课程或教育内容转化为具体行动步骤
- 创建学习任务和重复练习
- 生成学习计划

## 安装来源
- 仓库：softaworks/agent-toolkit
- 链接：https://github.com/softaworks/agent-toolkit

## 安装方式
```bash
npx skills add softaworks/agent-toolkit@ship-learn-next -g -y
```

## 指令
- `/ship-learn-next` - 将学习内容转化为行动计划
